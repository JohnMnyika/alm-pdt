## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd alm-pdt
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* [React](https://reactjs.org/)
* [Redux](https://redux.js.org/)
